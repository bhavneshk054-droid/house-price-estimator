# house-price-estimator

A Python-based application that estimates house prices using input parameters like area and number of bedrooms.

## 🚀 Features
- User input for area and bedrooms
- Calculates estimated house price
- Simple and efficient logic

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open the project in Python
2. Run the file:
   python main.py
3. Enter area and bedrooms when prompted

## 📌 Example
Input:
Area = 1200  
Bedrooms = 2  

Output:
Estimated Price = 3400000
